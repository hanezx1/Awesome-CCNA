📖 About

Welcome to CCNA Cheat Sheets.

This repository contains my personal CCNA study notes, Cisco IOS command references, and Packet Tracer cheat sheets created while preparing for the Cisco Certified Network Associate (CCNA) certification.

The goal is to provide a fast and organized reference for students, beginners, and networking enthusiasts who need quick revision material.

💻 Cisco IOS Commands

This repository includes quick references for frequently used Cisco IOS commands such as:

- show running-config
- show ip interface brief
- show vlan brief
- show interfaces trunk
- show spanning-tree
- show ip route
- show access-lists
- show ip ospf neighbor
- show ip nat translations

...and many more.
